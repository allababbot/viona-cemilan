# PT. VIONA CEMILAN Website

Website company profile untuk PT. VIONA CEMILAN, produsen Arai Pinang khas Sumatera Barat.

## Fitur
- Desain colorful dan modern dengan nuansa Minang.
- Responsif untuk mobile dan desktop.
- Animasi scroll yang interaktif.
- Galeri produk.

## Cara Menjalankan

1.  Pastikan Anda berada di folder proyek:
    ```bash
    cd viona-cemilan
    ```

2.  Install dependencies (jika belum):
    ```bash
    npm install
    ```

3.  Jalankan server development:
    ```bash
    npm run dev
    ```

4.  Buka browser di alamat yang muncul (biasanya `http://localhost:5173`).

## Catatan
- Gambar saat ini menggunakan placeholder karena keterbatasan kuota generate image. Anda dapat mengganti gambar di folder `public` atau mengubah URL di `index.html` dengan gambar asli Anda.
